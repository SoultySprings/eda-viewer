About This Viewer

Hi! I'm SoultySprings, and I built this interactive viewer for Jupyter Notebook (.ipynb) files using pure HTML, CSS, and JavaScript. I coded it with help from Google's Gemini and ChatGPT.

This project is a viewer for my own notebooks. It fetches raw .ipynb files from my GitHub, parses the JSON, and dynamically renders the markdown, code, and outputs. The design is neomorphic and includes a persistent light/dark theme.
